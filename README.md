# 새한패키지 견적서

## GitHub Pages 배포 방법

1. GitHub 로그인 → 우측 상단 **+** → **New repository**
2. Repository name: `saehan-quote` (공개: Public)
3. **Create repository** 클릭
4. **uploading an existing file** 클릭
5. 이 폴더의 파일 4개 모두 업로드:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
6. **Commit changes** 클릭
7. Settings → Pages → Branch: main → Save
8. 잠시 후 `https://[아이디].github.io/saehan-quote` 접속
9. 크롬/사파리에서 **앱 설치** 가능!
